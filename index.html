1. Proposito

El proyecto a presentar tiene como nombre "ClockMind", en este documento se describira la mayor parte de las funcionalidades del software

2. Alcance del producto / Software

El objetivo del proyecto es ayudar a las personas a organizar su vida. La mayoria de usuarios que utilizan aplicaciones de este tipo no tienen ningun tipo de contacto con la gente que tambien usa esas mismas aplicaciones, lo que buscamos es solucionar esta problematica, creando un espacio donde todos puedan comunicarse y "darse una mano" mutuamente.

3. Referencias

Para la creacion de este proyecto se ha tenido en cuenta la mayoria de aplicaciones similares sobre estilos de vida y rutinas, el objetivo es mejorarlas añadiendo la comunicacion entre usuarios.

4. Funcionalidades del producto

Las principales funcionalidades de ClockMind son:

  1. Crear y programar rutinas diarias y compartirlas con otros usuarios
  2. Reseñas rutinas para evaluar su efectividad y ranquearlas por estrellas
  3. Ver un ranking global de rutinas, que se podrá clasificar por semana, mes, año y siempre.
  4. Perfiles verificados de profesionales de la salud que pueden recomendar ciertas rutinas.
  5. Ganar puntos cada vez que se cumplan objetivos diarios y canjearlos por premium o decoraciones para el perfil.

5. Clases y caracteristicas de usuarios

En la aplicacion van a haber 4 tipos de usuarios:

  • Offline: Utiliza unicamente la seccion de hacer su propia rutina sin tener en cuenta la seccion de comunidad
  • Normal: Utiliza la seccion de comunidad para formar sus rutinas diarias
  • Creadores: Usuario que ademas que crea o mejora rutinas para compartirlas con el resto de usuarios
  • Profesionales: Usuarios que tienen una carrera completa y conocimientos sobre la salud

6. Entorno Operativo

La aplicacion al menos en su primera version, unicamente funcionara en dispositivos moviles con el sistema operativo Android.
